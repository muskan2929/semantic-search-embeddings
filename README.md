# 🚀 Semantic Search: From TF-IDF to Sentence-BERT

This project explores how text representation techniques have evolved from traditional sparse vectors to modern semantic embeddings. Using the **Quora Question Pairs** dataset, I compared multiple NLP models to understand how well they capture semantic similarity for real-world search applications.

## 🔍 Models Compared

* Bag of Words
* TF-IDF
* Word2Vec
* FastText
* Sentence-BERT

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Gensim
* Sentence-Transformers
* Pandas
* Matplotlib

## 📊 Key Results

* Compared model performance using **Accuracy, Precision, Recall, and F1 Score**
* Visualized embeddings with **PCA** and **t-SNE**
* Demonstrated why **Sentence-BERT** outperforms traditional frequency-based methods in semantic search

## 📌 Applications

* Semantic Search
* Duplicate Question Detection
* FAQ Retrieval
* Information Retrieval
* RAG (Retrieval-Augmented Generation)
